Quiz: Valid Final Scores

PROGRESS

In a women’s Grand Slam match (best of 3 sets), which two choices is a valid final score?

Correct! See explanation below.
Incorrect! Try again.

Explanation:

  • 6–0, 6–5 — Not valid. Must win a set by 2 games or via tiebreak at 7–6.
  • 2–6, 6–0 — Not valid. Match still ongoing; 1–1 in sets.
  • 6–2, 5–7, 6–3 — Valid. Winning 2 sets out of 3 ends the match.
  • 6–1, 6–2 — Valid. Straight set win, 2–0.
